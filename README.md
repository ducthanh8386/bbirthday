# Birthday Letter Web

Web chúc mừng sinh nhật tĩnh bằng HTML, CSS và JavaScript.

## Cách chạy

Mở `index.html` trực tiếp trong trình duyệt.

## Cá nhân hóa nhanh

- Sửa `CONFIG` trong `script.js` để đổi tên mặc định và lời chúc.
- Hoặc truyền qua URL:

```text
index.html?name=Minh&wish=Chuc%20ban%20sinh%20nhat%20that%20vui
```
